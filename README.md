📌 Overview

ShopEase is a full-stack web application that allows users to browse products, manage their cart, and place orders securely. It demonstrates real-world e-commerce functionalities using modern web technologies.

The project follows a client-server architecture with REST APIs and secure authentication.

⚙️ Installation & Setup
1️⃣ Clone the Repository
git clone https://github.com/kripalinigudadhe/ShopEase-.git
cd ShopEase-
2️⃣ Install Dependencies
npm install
3️⃣ Setup Environment Variables

Create a .env file and add:

PORT=5000
DB_URI=your_database_url
JWT_SECRET=your_secret_key
4️⃣ Run the Project
npm start
